## 3. ThirdMiniProject_SensorDataMonitoring_MQTT

### 요약
센서 데이터 실시간 조회 대시보드입니다.

### 사용된 센서
- DHT(온도 습도)
- BH1750(조도)

### 프로토콜
MQTT

### 사용된 기기
- Publisher & Subscriber: esp8266/NodeMCU 1.0(ESP-12E Module)
- Broker: RasberryPi

### 데이터베이스
InfluxDB

### 웹 대시보드
Grafana

## 4. FinalProject_PharmaceuticalMES

### 요약
